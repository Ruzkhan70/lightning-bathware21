# Lightning Bathware

Sri Lanka's Premier Hardware Store Web Application. Built with React 19, Vite 6, and Tailwind CSS 4.

## Features

- **Modern UI**: Clean, responsive design using Tailwind CSS 4 and Framer Motion.
- **Admin Dashboard**: Full control over products, categories, orders, and offers.
- **E-commerce**: Product catalog, cart, wishlist, and checkout.
- **Invoices**: PDF generation and verification.
- **Performance**: Lazy loading and optimized assets.
- **SEO**: Meta tags, robots.txt, and sitemap.

## Tech Stack

- **React 19**
- **Vite 6**
- **Tailwind CSS 4**
- **Firebase** (Firestore & Storage)
- **Radix UI** & **Shadcn UI**
- **Framer Motion**

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deploying

The app is optimized for deployment on Vercel.

```bash
vercel
```
